# FoodBridgeWeb
This web application facilitates the user to donate the food who are in need and it connects restaurants with NGOs to reduce food waste and fight hunger.

Features:

Restaurant Portal: Report surplus food and manage donations
NGO Portal: View available food donations and request pickups
Admin Dashboard: Monitor system activity and manage users
Real-time Updates: Instant notifications for new donations

How to Use

Restaurants:

Login to the restaurant portal
Add food waste details with photos
View NGO requests and approve/reject them
Track donation history

NGOs:

Login to the NGO portal
Browse available food donations
Request pickup for needed items
Track collection history

Admin:

Monitor system statistics
Approve new restaurant and NGO registrations
View analytics and reports

Deployment

This application is deployed using GitHub Pages and is accessible at: https://github.com/Malaiappan59/FoodBridgeWeb/

Technology Stack

HTML5, CSS3, JavaScript

Font Awesome for icons

Session Storage for user authentication

Bootstrap 5 for responsive design


Local Development

Simply open index.html in a web browser to run the application locally.
